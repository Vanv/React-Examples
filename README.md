# React Examples
React Examples for 
     - Display Simple text 
     - Props 
     - Event Handling etc 

Getting Started

- install all the necessary node modules
> npm install
     plus install all the node modules
        - babel-core
        - babel-loader
        - babel-preset-es2015
        - babel-preset-react
        - webpack
        - webpack-dev-server
   
to build 
> webpack -p

to run 
> npm start

Content is served from src/client/
