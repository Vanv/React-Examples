# React-Examples
React Examples for display text, Props, Event Handling etc 

Getting Started

- install all the necessary node modules
> npm install
     plus install all the node modules
        - babel-core
        - babel-loader
        - babel-preset-es2015
        - babel-preset-react
        - webpack
        - webpack-dev-server
   
to build 
> webpack -p

to run 
> npm start

All the source are inside src/client/app
